### Key Calculations

**Position Percentage**:
```
Position % = (Current Position Value / Total Portfolio Value) √ó 100
Example: $1,850 position / $10,280 portfolio = 18.0%
```

**Drawdown from Peak**:
```
Drawdown % = [(Current Value - Peak Value) / Peak Value] √ó 100
Example: ($875 current - $1,200 peak) / $1,200 = -27.1%
```

**Alpha vs S&P 500** (in basis points):
```
Alpha (bps) = (Portfolio Total Return % - S&P 500 Total Return %) √ó 100
Example: (3.45% - 1.58%) √ó 100 = +187 basis points
```
